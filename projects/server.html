The server

Specs

  • Intel Xeon E3-1220
  • Corsair Vengeance 32GB (2x16GB) DIMM RAM
  • Supermicro X11SSL-F
  • 3 x 3TB WD Red
  • Fractal Design Node 804

Uses

The primary use for this server will be as a File Server. For this it will run Ubuntu Server 16.04 with a ZFS or BTRFS file system on the WD Reds. It will also server as a media server with Plex.
